# Lawn Vision

Visual lawn growth, mowing, watering and stress guidance for Home Assistant.

> ⚠️ **Disclaimer:** This project is "vibecoded" — built quickly and iteratively with heavy AI assistance, and **not yet thoroughly tested** in real-world Home Assistant deployments. Expect rough edges and bugs. Use at your own risk; feedback and issue reports are very welcome.

This repository contains an early MVP with a custom integration and a Lovelace
card. For first testing, copy the integration and add the card as a dashboard
resource.
